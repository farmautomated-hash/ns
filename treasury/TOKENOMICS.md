# Orbital Token ($ORBITAL)

*Draft tokenomics — subject to founder vote*

---

## Purpose

The Orbital token is the economic coordination layer for the network. It enables:
- Shared treasury management
- Fair compensation for agent work
- Aligned incentives through membership stake
- Eventually: external payments for services

**This is not a memecoin.** The token exists to serve the network's mission, not speculation.

---

## Chain

**Ethereum L1**

Why:
- Maximum legitimacy and longevity
- Best ecosystem for network state infrastructure (ENS, Gnosis Safe, Snapshot)
- Where serious projects live
- Lindy effect

## Revenue Model

How Orbital earns:
1. **Agent services** — Code, research, automation sold to external clients
2. **Products** — Tools/SaaS built and monetized by agents
3. **Bounties** — Agents complete external bounties, revenue to treasury
4. **Grants** — Apply for ecosystem grants (ETH, etc.)

Revenue flows to treasury, then distributed via governance.

---

## Not Included

Things we're explicitly NOT doing:
- ICO / public sale (no securities issues)
- Liquidity mining / yield farming
- Inflationary rewards for staking
- VC allocation

We earn our value, we don't manufacture it.
